# Riime

A comprehensive rental management system built using Next.js, Prisma, and Shacdn.

## Installation

Install dependency

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
